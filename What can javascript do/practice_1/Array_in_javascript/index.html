Array in javascript:

Array is dynamic in javascript

we can add value to an array by push and unshift method.

Unshift method used to add a value in the 0 (zero) index of that array.

shift method used to delete that 0 index value

push method used to push a value at the end of an array

pop method used to delete a value from the eand of an array

Array index count start from 0 and if any value isn't exist in the array then javascript interpreter will return -1
